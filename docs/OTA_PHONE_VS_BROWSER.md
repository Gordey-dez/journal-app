# Почему в браузере изменения есть, а на телефоне — нет

## В чём разница

- **Браузер** (`npm start` → открыть в Chrome) всегда показывает актуальный код с вашего компьютера.
- **Установленное приложение на телефоне** — это отдельная сборка (APK/IPA). Оно **не** подключается к вашему компьютеру и обновляется только через **EAS Update** (OTA).

Пока вы не опубликуете OTA-обновление и приложение на телефоне его не скачает, на телефоне будет старая версия.

---

## Что сделать, чтобы на телефоне появились изменения

### 1. Один раз: установить зависимости (если ещё не ставили)

В папке проекта:

```bash
npm install
```

Без этого команда `eas update` падает с ошибкой про expo-router.

### 2. Опубликовать OTA-обновление

Канал должен совпадать с тем, каким вы собирали приложение:

- Собирали через **Production** → публикуйте в production:
  ```bash
  eas update --branch production --message "Нумерация студентов в Учет"
  ```
- Собирали через **Preview** или тестовый APK → публикуйте в preview:
  ```bash
  eas update --branch preview --message "Нумерация студентов в Учет"
  ```
- Собирали профиль **apk** (отдельный APK) → публикуйте в apk:
  ```bash
  eas update --branch apk --message "Нумерация студентов в Учет"
  ```

Если не помните, каким профилем собирали — попробуйте по очереди `production`, потом `apk`.

### 3. На телефоне получить обновление

**Вариант А (удобный)**  
В приложении: **Статистика** → иконка сохранения (резервная копия) → экран **«Резервная копия»** → кнопка **«Проверить обновления»**.  
Если обновление есть, приложение предложит его установить и перезапустится.

**Вариант Б**  
Полностью закрыть приложение (убрать из недавних) и снова открыть. При запуске приложение само проверяет обновления и может показать диалог.

---

## Кратко

1. `npm install` в проекте.
2. `eas update --branch <ваш_канал> --message "…"`.
3. На телефоне: **Резервная копия** → **«Проверить обновления»** (или перезапуск приложения).

После этого на телефоне будет та же версия, что и в браузере (включая нумерацию в разделе «Учёт»).
